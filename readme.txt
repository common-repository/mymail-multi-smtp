== Description ==

Allows to use multiple SMTP connection for the MyMail Newsletter Plugin

== Installation ==

1. Upload the entire `mymail-multi-smtp` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings => Newsletter => Delivery and select the `Multi SMTP` tab
4. Enter your credentials
5. Send a testmail
